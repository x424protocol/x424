# Security policy

No release is production-supported yet. x424 is pre-alpha and unaudited.

Do not open public issues for suspected vulnerabilities. Use the GitHub
repository's private vulnerability-reporting feature once enabled, or contact
the repository owner privately.

Include the affected component, reproduction, expected impact, and mitigation.
Never include real World/provider proofs, nullifiers, result tokens, private
keys, pairwise secrets, recovery material, production subjects, or user data.

The full threat model is in [`docs/SECURITY.md`](docs/SECURITY.md).
