# x424.org

Minimal public landing page for the [x424 Human Dependency Protocol](https://github.com/x424protocol/x424).

```bash
npm install
npm test
```

The site is deployed through OpenAI Sites and uses no database, storage,
authentication, or custom client-side interaction. Google Analytics 4 is enabled for
basic pageview and enhanced-measurement reporting; it does not send user IDs or
custom application events.
